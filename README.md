# pv-nodejs-layer-7
About PV Node L7 is a Node.js tool for Layer 7 load testing &amp; HTTP/2 security audits. Supports HTTP/1.1, 2, &amp; 3 with realistic traffic, randomized TLS, &amp; adaptive delays. Includes Rapid Reset &amp; MadeYouReset attacks, real-time CLI stats, &amp; concurrency control. Needs Node.js v16+ &amp; npm. For authorized testing only.
